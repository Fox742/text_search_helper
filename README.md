# TextSearchHelper

Библиотека, позволяющая осуществлять высокопроизводительный полнотекстовый поиск в файле. Написана мной для тренировки после того, как я столкнулся с подобной задачей в моей рабочей практике, в рамках которой программа должна была осуществлять полнотекстовый поиск в большом текстовом файле. Так процедуру поиска необходимо было осуществлять внушительное количество раз, поэтому стояла задача оптимизации полнотектового поиска.
